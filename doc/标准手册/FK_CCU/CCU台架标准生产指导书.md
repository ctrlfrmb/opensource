## 3. 术语与定义
- CCU台架：用于CCU软硬件测试的机柜化测试平台。
- DUT：被测件（客户CCU或相关样件）。
- NG：不合格；PASS：合格；NA：不适用。
- 关键件：影响安全、上电、通信、射频链路的器件与线束。

## 4. 职责分工
- 安装工人：按图纸/BOM完成装配、布线、标签、安装自检。
- 测试工程师：按检验表完成功能测试、记录证据、问题复测。
- 项目经理：组织计划、风险、放行评审与交付签字。
- 质量人员：对关键节点抽检与放行复核。

## 5. 流程总览
1. 生产准备
2. 机柜与模块安装
3. 线束制作与布线
4. 安装自检
5. 测试执行
6. 问题闭环与复测
7. 交付评审与放行

## 6. 安装工位作业指导（安装工人执行）

### 6.1 开工前确认
1. 核对项目信息：项目号、项目名称、产品编号、计划日期。
2. 核对资料版本：BOM、接线定义、检验表版本一致。
3. 核对物料状态：关键件必须齐套，不齐套不得开工。
4. 核对工具状态：压接、测量、标签打印工具可正常使用。

### 6.2 安装步骤
1. 安装机柜及电气部件（机柜、PDU、UPS、电气箱）。
2. 安装模块与仪器（负载箱、LVDS箱、射频切换箱、交换机、工控机、CMW100等）。
3. 按接线资料完成强弱电布线，强弱电分离。
4. 所有设备接地线接入机柜接地铜排。
5. 按低压线束/同轴线束定义完成端接与连接。
6. 按线束标签模板打印并粘贴双端标签。

### 6.3 安装质量要求
- 固定可靠：螺丝无缺失、无松动。
- 走线规范：线槽内布线整齐、无悬空、无锐边磨损。
- 端接合格：无裸铜、无虚焊、无反接、无缩PIN。
- 可维护性：标签可读，接口可触达，不影响机柜门开关。

### 6.4 安装自检项目（必须完成）
- 外观检查：机柜与模块无损伤、无污渍、无异物。
- 电气检查：漏保/断路器/急停配置正确。
- 接地检查：设备接地完整、连续性正常。
- 通信基础检查：关键模块可上电、可识别。
- 线束检查：名称、去向、长度与表单一致。

### 6.5 安装工位输出记录
- FRM-01《安装完成记录单》
- FRM-02《线束标签与去向对照表》
- 《缺料/替代料清单》（如有）

## 7. 测试工位作业指导（测试工程师执行）

### 7.1 测试前条件（Gate）
1. 已收到安装工位输出记录，且签字完整。
2. 供电、网络、测试电脑、仪器状态正常。
3. 检验模板使用最新版，项目号/产品编号已填写。
4. 安全条件确认完成（急停有效、保护器件状态正常）。

### 7.2 测试执行顺序
1. 机柜检验（`机柜--检验表`）
2. LVDS检验（`LVDS--检验表`）
3. A2B检验（`A2B--检验表`）
4. 模拟量采集模块检验（`模拟量采集模块--检验表`）
5. 射频切换箱检验（`射频切换箱--检验表`）
6. RF线损检验（`RF--线损检验表`）

### 7.3 关键测试要求
- LVDS：按检验表IP逐项验证出图并留图像证据。
- LVDS：须按检验表要求完成板卡IP可达性确认，使用客户B样进行图像验证并贴图。
- A2B/模拟量：逐项验证link与数据传输结果。
- 射频切换箱：逐通道验证继电器切换和通断定义。
- RF线损：按检验表频点记录结果；每频点应可读取数值且线损不超过20dB（以检验表为准）。
- RF线损：按检验表要求留存CMW100测试截图与测量证据。
- 长稳测试：完成12h硬件资源压测和12h带DUT压测（项目若裁剪须书面批准）。

### 7.4 测试记录要求
- 每项必须记录：对象、步骤、结果、测试人、时间、证据。
- 结果只能填写PASS/NG/NA，不得空项。
- NG必须开问题单，记录临时措施、根因、责任人、复测结论。

### 7.5 测试工位输出记录
- FRM-03《测试执行记录单》
- 《机柜检验表（签字版）》
- 《模块检验表（签字版）》
- 《RF线损测试记录》
- FRM-04《问题闭环与复测记录》

## 8. 项目管理与交付（项目经理执行）

### 8.1 里程碑控制
- M1：资料冻结（BOM/接线/检验表）
- M2：物料齐套
- M3：安装完成并自检通过
- M4：测试完成并闭环
- M5：交付评审通过并放行

### 8.2 放行条件
- 必测项全部PASS或经批准NA。
- 所有NG项已闭环并完成复测。
- 交付资料齐全且版本一致。
- 相关责任人签字完整。

### 8.3 交付资料清单
- 最终BOM
- 接线定义与线束清单
- 全套检验记录与测试报告
- 问题闭环记录
- 发货拍照记录与发货清单
- FRM-05《交付评审与放行单》
- FRM-06《客户验收签收单》

## 9. 异常处理
- 发现关键件缺失：停止作业，升级项目经理。
- 发现安全风险：立即断电并上报。
- 发现图纸/BOM冲突：暂停执行，待变更确认后恢复。
- 发现测试异常：先隔离问题，再复测确认，不得跳项放行。

## 10. 安全与红线
- 未完成通电前检查，禁止上电。
- 未通过关键测试，禁止交付。
- 未完成NG闭环，禁止签署通过。
- 文档与实物不一致，禁止出货。

## 11. ZCU台架复用要求（后续项目）

- 复用表单：FRM-01~FRM-06保持不变。
